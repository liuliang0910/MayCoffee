# 微信通知功能设置指南

这个网站现在支持在有客户留言或回复时，通过微信通知你。**完全免费！**

## 工作原理

使用 **Server酱** 服务，将网站的通知发送到你的微信。

## 快速设置（只需3步）

### 第1步：获取 Server酱 SCKEY

1. 用微信打开这个链接：https://sct.ftqq.com/
2. 点击"登录"，用微信扫码登录
3. 登录后，你会看到你的 **SCKEY**（看起来像：`SCU123456789abcdefg...`）
4. **复制这个 SCKEY**

### 第2步：填入配置

打开 `app.py` 文件，找到第19-21行：

```python
SERVERCHAN_CONFIG = {
    'sckey': 'YOUR_SCKEY'  # 替换为你的 Server酱 SCKEY
}
```

把 `YOUR_SCKEY` 替换为你刚才复制的 SCKEY。

例如：
```python
SERVERCHAN_CONFIG = {
    'sckey': 'SCU123456789abcdefg'
}
```

### 第3步：安装依赖

在你的项目文件夹中运行：

```bash
pip install -r requirements.txt
```

## 完成！

现在当有客户留言或回复时，你会在微信上收到通知。

## 常见问题

### Q: 为什么没有收到微信通知？

A: 检查以下几点：
1. 确认你的 SCKEY 是否正确填入 `app.py`
2. 确认你在 Server酱 网站上用微信扫码登录过
3. 检查你的网站是否正确运行
4. 查看 Python 控制台是否有错误信息

### Q: Server酱 是什么？

A: Server酱 是一个免费的消息推送服务，可以把服务器的消息推送到你的微信。官网：https://sct.ftqq.com/

### Q: 这个服务稳定吗？

A: Server酱 是一个比较稳定的服务，已经运行多年。如果你想要更稳定的方案，也可以考虑钉钉或企业微信。

### Q: 可以改成钉钉或企业微信吗？

A: 可以的，原理是一样的，只需要改一下 API 调用部分。如果需要，可以告诉我。

### Q: 如何测试通知功能？

A: 
1. 确保配置正确
2. 在网站上提交一条留言
3. 检查你的微信是否收到通知
4. 如果没收到，查看 Python 控制台的错误信息

## 需要帮助？

如果有问题，请查看 Python 控制台的输出。成功发送通知时会显示 "✅ 微信通知已发送"，失败时会显示 "❌ 微信通知发送失败" 和错误信息。
